User

Searching for the metabolites present in a certain oil:

Select an oil from dropdown menu under oil names.

Then Press Submit to get a list of all the metabolites present in that oil.

Searching for the oils in which a certain metabolite is present:

Select a metabolite from dropdown menu under metabolite names.

Then Press Submit to get a list of all the oils in which that metabolite is present.

Admin

Admin needs to log in to the application to do the following things:

Add data of a new oil using a csv file:

Fill all the required feilds and upload the .csv file. Click submit to add the data to the database.

Adding new admins:

This can be done under the Authentication and Authorization menu.

Changing Specefic data already present in the database:

This can be done under the APP menu. Where a metabolite or an oil can be deleted. Through menu can be used to change the attributes of the through table
